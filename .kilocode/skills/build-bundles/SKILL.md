# Сборка AssetBundle

## Инструкция
1. Убедись, что все нужные бандлы уже назначены (можно спросить у пользователя).
2. Выполни сборку через MCP-инструмент `manage_build` с действием `build`, указав выходную папку `BuiltBundles` (в корне проекта).
3. После сборки сообщи пользователю:
   - Полный путь к папке `BuiltBundles`.
   - Структуру папок и файлов (через `list_directory` рекурсивно).
   - Размер каждого файла (если доступно `get_file_info`).
4. Предложи открыть папку.
