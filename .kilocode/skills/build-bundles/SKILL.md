# Сборка AssetBundle

## Инструкция
1. Убедись, что все нужные бандлы уже назначены (можно спросить у пользователя подтверждение).
2. Выполни сборку через MCP-инструмент `build_asset_bundles` с выходом в папку `BuiltBundles` (в корне проекта).
3. После сборки сообщи пользователю:
   - Путь к папке с бандлами (`BuiltBundles`).
   - Размер каждого файла (можно получить через MCP `get_file_info` или просто прочитать из проводника, но лучше, если MCP поддерживает, иначе сообщи пользователю, где смотреть).
4. По желанию предложи пользователю открыть папку с бандлами.
