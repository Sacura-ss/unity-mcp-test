# Создание AssetBundle

## Инструкция
1. Спроси пользователя, для какой папки нужно создать бандл (например, `Art/Environment` или `Art/UI`).
2. С помощью MCP-инструмента `assign_asset_bundle` назначь бандл с именем `{имя папки}_bundle` (например, `env_bundle`).
3. Если папка ещё не создана — создай её через MCP.
4. **Назначь AssetBundle для всех текстур в папке**: для каждого файла в целевой папке используй `manage_asset` с действием `modify` и свойством `assetBundleName`, установив имя бандда (например, `characters_bundle`).
5. После назначения бандлов предложи пользователю выполнить сборку.
6. Выполни сборку через MCP-инструмент `build_asset_bundles` с выходом в папку `BuiltBundles`.
7. Сообщи пользователю о результате и пути к собранным бандлам.
