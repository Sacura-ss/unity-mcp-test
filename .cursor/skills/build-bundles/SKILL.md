# Сборка AssetBundle

## Инструкция
1. Проверь, что все бандлы назначены.
2. Выполни сборку через MCP-инструмент `build_asset_bundles` с выходом в папку `BuiltBundles`.
3. Сообщи пользователю о результатах и размерах файлов.
